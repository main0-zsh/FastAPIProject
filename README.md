# FastAPI Feedback System 💬

Простой и эффективный API-сервис для сбора и хранения пользовательских отзывов. Проект демонстрирует работу с асинхронными запросами в FastAPI и интеграцию с базой данных.

## 🚀 Основные функции
* **Отправка отзывов:** Пользователи могут оставлять свои отзывы через POST-запросы.
* **Сохранение в БД:** Все данные (имя пользователя, текст отзыва, рейтинг) надежно записываются в базу данных.
* **Получение списка:** Возможность просмотреть все оставленные отзывы через GET-запрос.
* **Валидация данных:** Автоматическая проверка входящих данных с помощью Pydantic.

## 🛠 Стек технологий
* **Python 3.12**
* **FastAPI** (современный и быстрый веб-фреймворк)
* **SQLAlchemy** / **SQLModel** (для работы с БД)
* **Pydantic** (валидация данных)
* **SQLite** (локальная база данных)

## 📦 Как запустить проект

1. **Клонируйте репозиторий:**
   ```bash
   git clone [https://github.com/main0-zsh/FastAPIProject.git](https://github.com/main0-zsh/FastAPIProject.git)
   cd FastAPIProject
Настройте окружение и зависимости:

Bash
python3 -m venv .venv
./.venv/bin/python3 -m pip install fastapi uvicorn sqlalchemy
Запустите сервер:

Bash
./.venv/bin/python3 -m uvicorn main:app --reload

https://www.google.com/search?q=http://127.0.0.1:8000/docs — здесь можно протестировать отправку отзывов прямо из браузера.
